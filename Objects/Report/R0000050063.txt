         Captions have been updated for Help File purposes

         NOTE: The Report Layout page size dimensions of the 'Report' element should match the printer
          label size dimensions that are set for the printer.
         - Tested with the Zebra GK 420t Label Printer
         - This is a 3x2 label, ensure the label's actual paper size is also 3x2 for proper printing.
         - If issues such as printing across two labels persisit, to change "Orientation" from portrait to landscape:
                View-->Layout from Designer, and then Right-Click-->Report Properties within the RDLC report.

         NOTE: The font size for the 3 of 9 Barcode has been set to 28 and can't be set smaller than that
          because the label printer does not print clear spaces between barcode lines when the font is smaller
          than 28.  This means that even though "Item No" is of type code(20), this label will only work with
          Item No's that have a length of 15 or less.  Anymore, and the code will "wrap" to the next line.
       </REVISION>
